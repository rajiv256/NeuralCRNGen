# Neural CRNs

This repository contains code for running the simulations presented in the paper [Neural CRNs: A Natural Implementation of Learning in Chemical Reaction Networks](https://doi.org/10.48550/arXiv.2409.00034)

## Branches for different experiments

- [Linear Regression](https://github.com/rajiv256/NeuralCRNGen/tree/linear_regression) 
- [Nonlinear Regression ($y = x_1 x_2 + x_2^2$)](https://github.com/rajiv256/NeuralCRNGen/tree/nonlinear_regression_z2_dotprod) 
- [Nonlinear Regression ($y = \sin(x_1) + x_2^2)](https://github.com/rajiv256/NeuralCRNGen/tree/nonlinear_regression_z2_dotprod_sinxx2) 
- [Nonlinear Classification (cubic $f_\theta$)](https://github.com/rajiv256/NeuralCRNGen/tree/nonlinear_z3) 
- [Nonlinear Classification (quadratic $f_\theta$)](https://github.com/rajiv256/NeuralCRNGen/tree/nonlinear_z3_linapprox)
